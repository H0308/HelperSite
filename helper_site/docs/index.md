## 使用指导

本网页提供两种主题，新版主题中包含一些动画和特效，对浏览器性能要求较高，建议在Chrome桌面端浏览器下使用；旧版主题去除了部分动画和特效，并且为了保证阅读性，对背景也有所改动，建议在移动端浏览器和Edge等性能稍弱的浏览器中使用

对于上文未提到的浏览器可自行尝试哪一种主题更适合阅读并且可以流畅运行

<span style="color:red">注意，切换主题只是改变网页显示效果，并不会改变网页本身对内存的占用</span>
