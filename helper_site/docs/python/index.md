# Python章节目录

## Python基础知识

本节介绍了Python中的基础语法，包括注释、基本的输入和输出函数、数据类型、运算符、分支和循环语句

## Python组合数据类型

本节介绍了Python中常见的组合类型及基本使用，包括序列、元组、`range`、字典和集合

## Python异常处理

本节基本介绍了Python中的异常处理机制

## Python函数

本节介绍了Python中定义函数的方式，另外还介绍了函数中的默认参数、可变类型和不可变类型、关键字参数、可变参数。除了这些内容以外，本节还介绍了回调函数、函数作为返回值、Python中的作用域问题、闭包、Lambda表达式、注解和装饰器

## Python模块

本节介绍了Python中模块和包的概念和使用注意

## Python输出与文件操作

本节首先详细介绍了Python中的输出函数`print`，再介绍了格式化字符串字面值。接着，本节还介绍了Python中的文件操作

## Python类和对象基础

本节首先介绍了Python中如何创建类以及实例化类对象，接着从面向对象三大特性（封装、继承、多态）的角度介绍它们在Python中的体现。最后，本节还介绍了Python中的迭代器和生成器

## Python类和对象进阶

本节主要介绍了Python中关于类和对象的一些补充内容，包括限制属性绑定、使用`@property`装饰器、多重继承、魔法方法
